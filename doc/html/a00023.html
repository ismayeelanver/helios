Detailed Description

The namespace which holds all the structures and functions neccessary to generate a config for the WM.

typedef struct WMConfig::Action WMConfig::Action

This struct represents an action that can be performed when a key is pressed.

It contains the type of the action and the target of the action.

◆ ActionType

typedef enum WMConfig::ActionType WMConfig::ActionType

@breif This enum represents the type of action that can be performed when a key is pressed.

It can be one of the following:

  • run: Run a command.
  • ch: Change the currently focused window to the window with the specified target.
  • focus: Focus the window with the specified target.
  • close: Close the currently focused window.
  • toggle: Toggle the visibility of the window with the specified target.
